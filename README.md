# BossSpider Web Rewrite Workspace

This folder is a standalone copy of the BOSS crawler runtime and Web API scaffold.

It does not import files from the parent project. The copied crawler package lives in `new/crawler`, and role projects live in `new/projects`.

## Run API

```bash
pip install -r requirements.txt
python -m uvicorn backend.app:app --reload --port 8000
```

Open API docs:

```text
http://127.0.0.1:8000/docs
```

## Run Web UI

Frontend dependencies are restored from `bossspider-web/package-lock.json`.
Do not commit `node_modules`; it is platform-specific and should be generated locally.

```bash
cd bossspider-web
npm ci
npm run dev -- --host 127.0.0.1 --port 5173
```

Open:

```text
http://127.0.0.1:5173/
```

## Build Web UI

`dist/` is generated by the build step and is ignored by git.

```bash
cd bossspider-web
npm ci
npm run build
```
